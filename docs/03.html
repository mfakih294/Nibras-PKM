Records and Data Model

Certain fields are common to all module records: * Summary * Description * Priority * Type * Bookmarked * Date created and last updated

Other fields are particular to certain modules: * Start and end dates (journal and planner). * Completed date, planned and actual duration, percent completed (for tasks and goals). * Author, ISBN, published on, publication city, etc. (for resources). * Blog (for writings and notes).

Departments and courses

Department and course can assign any record.

Comments and tags

A comment can be added to any module record.

And any module record can have one or more tags.

You can see all records tagged with a certain tag from the Tags panel in the west region.
